# [Ingénieur.e en Développement Full Stack](https://sii-group.com/fr-FR/offers/developpement/ingenieure-en-developpement-full-stack)

## Contexte

Ce stage s’intègre à l’équipe en charge du développement des démonstrateurs pour nos laboratoires R&D, répartis sur plusieurs agences de SII. Ces démonstrateurs illustrent des travaux de recherche sur des sujets variés : IoT, interfaces, systèmes embarqués, sécurité, etc.  

## Objectifs de la mission

Le stage a pour objectif le développement d’application de démonstration pour nos projet de R&D. Ces démonstrateurs s’intègrent dans des projets de recherche sur des sujets variés (transport, ingénierie, IA, énergie, …) et se basent sur socle technique commun utilisant Java, Sprint Boot et Angular et/ou React. L’objectif du stage est d’intervenir dans le développements de ces applications sur l’ensemble des aspects (back end, front end, système) et dans un cadre méthodologique agile. 

# Profil recherché

Vous êtes issu(e) d’une École d’Ingénieur en dernière année d’études ou d’une formation universitaire Bac+5. Vous maîtrisez le développementen java avec Spring Boot ainsi que Angular, et vous avez des connaissances sur docker et le déploiement dans les environnements cloud comme AWS
